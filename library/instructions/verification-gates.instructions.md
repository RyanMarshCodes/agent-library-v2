# Verification Gates (Universal)

## Definition of Done

A task is done only when all applicable gates pass.

## Gates

1. Functional gate: behavior matches acceptance criteria.
2. Safety gate: no obvious regression introduced.
3. Quality gate: style and structure align with project norms.
4. Validation gate: relevant tests/build checks run successfully.
5. Traceability gate: summary includes changed files and rationale.

## Evidence

1. List checks executed.
2. List key outputs and failures if any.
3. Call out residual risk explicitly.
