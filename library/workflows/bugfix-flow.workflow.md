# Workflow: Bugfix Flow

## Purpose
Fix a defect with reproducible verification and low regression risk.

## Required Inputs

1. Bug report or failing behavior.
2. Reproduction steps.
3. Affected scope.

## Steps

1. Reproduce issue.
2. Isolate root cause.
3. Implement minimal fix.
4. Add or update regression test.
5. Verify fix and no obvious breakage.
6. Document root cause and prevention note.

## Required Outputs

1. Root cause summary.
2. Fix summary.
3. Regression validation evidence.
